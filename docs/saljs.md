# Sal.js

## Overview
Performance-focused, lightweight scroll animation library (vanilla JS).

## Install
- npm: `npm install sal.js`

## CDN
```html
<link rel="stylesheet" href="https://unpkg.com/sal.js/dist/sal.css">
<script src="https://unpkg.com/sal.js/dist/sal.js"></script>
```

## Usage
```html
<div data-sal="fade" data-sal-duration="600">Hello</div>
<script>sal({ once: true });</script>
```
