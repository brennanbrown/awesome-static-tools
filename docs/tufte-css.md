# Tufte CSS

## Overview
Beautiful sidenote styling inspired by Edward Tufte.

## CDN
```html
<link rel="stylesheet" href="https://edwardtufte.github.io/tufte-css/tufte.min.css">
```

## Usage
- Use `.sidenote` and `.marginnote` classes per docs.

## 11ty Tips
- Shortcodes for sidenote markup patterns.

## Accessibility
- Ensure sidenotes are readable in mobile.
