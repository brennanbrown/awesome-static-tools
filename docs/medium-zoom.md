# medium-zoom

## Overview
Medium-style image zoom effect.

## Install
- npm: `npm install medium-zoom`

## CDN (ESM)
```html
<script type="module">
  import mediumZoom from 'https://unpkg.com/medium-zoom@latest/dist/medium-zoom.esm.js';
  mediumZoom('[data-zoomable]');
</script>
```

## Accessibility
- Provide alt text; avoid obstructing captions.
