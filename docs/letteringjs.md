# Lettering.js

## Overview
Fine-grained control over typography by wrapping letters/words/lines in spans for advanced styling.

## CDN (requires jQuery)
```html
<script src="https://unpkg.com/jquery@3.6.4/dist/jquery.min.js"></script>
<script src="https://unpkg.com/letteringjs@0.7.0/jquery.lettering.js"></script>
```

## Usage
```html
<h1 class="title">Fancy Title</h1>
<script>
  $(function(){ $('.title').lettering(); }); // Wraps each letter
</script>
```

## Notes
- Consider performance when applying to large bodies of text.
