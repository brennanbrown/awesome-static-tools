# ez-tip

## Overview
Zero-dependency JavaScript tooltip library using CSS custom properties for styling.

## Install
- No build required; include the script directly from the repo or package if available.

## Usage
```html
<button class="has-tip" data-tip="Hello!">Hover me</button>
<script>
  // See project README for initialization patterns; many use CSS-only with [data-tip]
</script>
```

## Styling
- Customize via CSS variables (see project docs).
