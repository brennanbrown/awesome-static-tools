# Prism.js

## Overview
Elegant syntax highlighting for code blocks.

## Install
- npm: `npm install prismjs`

## CDN
```html
<link rel="stylesheet" href="https://unpkg.com/prismjs/themes/prism.min.css">
<script src="https://unpkg.com/prismjs/prism.js"></script>
```

## Usage
- Add `language-js` (etc.) classes to `<code>` blocks.

## 11ty Tips
- Use `markdown-it-prism` or `@11ty/eleventy-plugin-syntaxhighlight`.

## Performance
- Load only necessary languages.
